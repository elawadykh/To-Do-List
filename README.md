# 🚀 Real-Time To-Do List Application

موقع لإدارة المهام اليومية (To-Do List) مبني بـ JavaScript العادية ومربوط بقاعدة بيانات سحابية حقيقية لتحديث البيانات لحظياً.

---

## ✨ المميزات (Features)
* 🔐 **قاعدة بيانات سحابية:** ربط كامل مع **Firebase Firestore** لحفظ المهام أونلاين.
* ⚡ **تحديث لحظي (Real-time):** إضافة ومسح المهام بيسمع في الداتابيز في جزء من الثانية بدون ما الصفحة تعمل Reload.
* 📱 **تصميم متجاوب:** واجهة مستخدم مريحة للعين ومتناسقة مع الشاشات المختلفة (Responsive UI).
* 🔄 **إستضافة سحابية:** المشروع مرفوع لايف على سيرفرات **Firebase Hosting**.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Frontend:** HTML5, CSS3, JavaScript 
* **Backend/Database:** Firebase Firestore
* **Hosting:** GitHub Hosting
* **Version Control:** Git & GitHub
